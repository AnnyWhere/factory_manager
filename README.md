# Решенный билет экзамена ОП.14

### Содержание билета:
Задания 10
1.	Сериализация и десериализация. Файлы json
2.	Спроектировать классы: Работник, Цех, Завод. Унаследовать классы: Рабочий, Начальник цеха от класса Работник. Унаследовать классы: Токарь, Слесарь, Фрезеровщик от класса Рабочий. Построить диаграмму классов. Для хранения использовать списки.
3.	Написать программу, демонстрирующую работу с классами. Переопределить операции добавления, удаления, сравнения. Подразделения сравниваются по количеству работников одного класса. Данные и лог хранить в json-файлах. Использовать меню.
4.	Структурные паттерны проектирования. Фасад (Facade)

### Описание

Представляет собой проект factory manager, в котором раскрываются темы:

1. Сериализация и десериализация. Файлы json
2. Наследование
3. Перегрузка операторов
4. Структурный паттерн проектирования "Фасад"
5. Абстрактные классы и методы

А также теоретический материал (см. README в папке exam).

Так же здесь я оставлю свой мини-проект, который хорошо раскрывает тему json.
