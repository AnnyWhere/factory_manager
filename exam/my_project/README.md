# Менеджер заметок с JSON

Мини-проект, демонстрирующий работу с JSON-файлами в Python. Консольное приложение для управления заметками с сохранением данных в JSON-формате.

## Функциональность

- **Создание заметок** с автоматической генерацией ID
- **Чтение всех заметок** из JSON-файла
- **Удаление заметок** по ID
- **Форматированный вывод** в консоль

## Структура данных

Данные хранятся в файле `notes.json` в следующем формате:

```json
{
  "notes": [
    {
      "id": 1,
      "title": "Заголовок заметки",
      "content": "Текст заметки",
      "created": "2024-01-15 14:30",
      "tags": ["тег1", "тег2"]
    }
  ],
  "last_id": 1
}
```

## Использование

### Запуск приложения:
```bash
python main.py
```

### Доступные команды:
1. Показать все заметки
2. Показать заметку по ID
3. Добавить новую заметку
4. Удалить заметку

## Технические детали

### Основные компоненты:
- `NotesManager` - основной класс для управления заметками
- `load_notes()` - загрузка данных из JSON-файла
- `save_notes()` - сохранение данных в JSON-файл
- `display_note()` - форматированный вывод одной заметки
- `display_all()` - вывод списка заметок
- `console_control()` - консольное управление
- `delete_note()` - удалить заметку

### Особенности реализации:
- Автоматическое создание файла данных при первом запуске
- Поддержка кириллицы (UTF-8 кодировка)
- Читаемое форматирование JSON-файла
- Обработка ошибок ввода пользователя

## Требования

- Стандартная библиотека Python (модули: json, os, datetime)
